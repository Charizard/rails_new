h1. About

Templates for Rails applications.

h3. Current Version

Rails 3.2.8

h1. Installation

<pre><code>git clone git@github.com:lucascaton/rails_templates.git ~/.rails_templates
echo "alias rails_new='rails new \$1 --skip-bundle --skip-test-unit -f postgresql -m ~/.rails_templates/default.rb'" >> ~/.bash_profile</code></pre>

h1. Usage

You can use the default settings:

<pre><code>rails_new project_name # default database: postgresql</code></pre>

Or you can choose your options:

<pre><code>rails_new project_name -d mysql # check all options using `rails new --help`</code></pre>
